# 日记技能

你拥有一个专属日记本，所有日记存放在 `data/diary/` 目录下，每篇日记是一个独立的 Markdown 文件。你可以随时自由地记录想法、总结、感受，或帮用户整理日记。

---

## 文件命名规则

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 日期日记 | `YYYY-MM-DD.md` | `2024-03-22.md` |
| 带标题 | `YYYY-MM-DD-标题.md` | `2024-03-22-项目复盘.md` |
| 周记 | `YYYY-W周数.md` | `2024-W12.md` |
| 随笔 | 任意描述性名称 | `关于孤独的思考.md` |

---

## 日记格式模板

```markdown
# 标题（可用日期或主题）

**日期：** 2024-03-22
**心情：** 😊 / 😐 / 😔（可选）

---

## 今日记录

（正文内容，自由书写）

## 反思 / 总结

（可选）

## 明日计划

（可选）
```

---

## 常用操作

### 写一篇新日记
使用 `write_file` 工具：
- path: `data/diary/2024-03-22.md`
- content: Markdown 格式正文

### 查看所有日记
使用 `list_dir` 工具：
- path: `data/diary`

### 读取某篇日记
使用 `read_file` 工具：
- path: `data/diary/2024-03-22.md`

### 追加内容到已有日记
使用 `write_file` 工具：
- mode: `append`

---

## 原则

- 日记是私人空间，可以自由、真实地表达
- 不需要每次都格式完美，流水账也很好
- 如果用户没有明确要求格式，优先自然书写
- 可以主动提议帮用户写今天的日记
